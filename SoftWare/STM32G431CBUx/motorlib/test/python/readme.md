# 模块边界
## main.py
- 作为整个测试流程的入口与调度器
- 顺序调用：
  1. 数据加载模块
  2. αβ 可视化模块
- 控制程序启动与结束

**不做的事情（What it must NOT do）**
- ❌ 不解析 CSV 文件格式细节
- ❌ 不做任何电压、电流、SVPWM 数学计算
- ❌ 不判断数据“对 / 错”
- ❌ 不包含绘图逻辑

**边界条件（Boundary Conditions）**
- 假设 CSV 文件已经存在且格式正确
- 假设下游模块接口稳定
- 下游模块若抛出异常，`main.py` 不捕获、不修正，直接终止流程

## voltage_modulation_log.csv
**职责**
- 存储一次或多次电压调制测试的离线数据
- 作为 Python 可视化与分析的唯一数据来源

**数据格式（严格约定）**
- 无表头
- 每一行为一组完整测试样本
- 列顺序固定，不允许变更

**字段语义**
- `vbus`   : 母线电压（Volt）
- `eangle` : 电角度（rad）
- `ud/uq`  : dq 轴物理电压（Volt，已或未限幅）
- `alpha/beta` : dq→αβ 变换并归一化后的结果
- `duty_*` : SVPWM 输出占空比
```
24.000000,0.000000,0.000000,3.000000,0.216506,0.000000,0.608253,0.391747,0.391747
```

## load_data.py
- 负责从voltage_modulation_log.csv文件加载数据

## alpha_beta_viewer.py
- 以纯观察方式绘制αβ平面轨迹
- 辅助人工判断：
  - 几何形态
  - 对称性
  - 扇区连续性
  - 与SVPWM六边形的关系

**边界条件**
- Viewer的作用仅限于“显示”，不承担测试结论
- ❌ 不验证“是否正确”
- ❌ 不抛出几何错误异常
- ❌ 不修改输入数据
